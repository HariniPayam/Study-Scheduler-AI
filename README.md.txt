#AI-POWERED STUDY SCHEDULER

-This is a smart study planner app built using Streamlit and Python.  
-It helps students organize their daily study time by automatically scheduling subjects based on importance and total available hours.

---

## FEATURES

- Enter subjects and prioritize the most important ones
- Set how many hours you can study in a day
- Get a personalized, balanced study schedule in seconds
- Simple and clean web interface using Streamlit

---

## TECH STACK

- Python 3.12  
- Streamlit  
- GitHub for version control

---

## HOW TO RUN

1. Make sure you have Python installed
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
3. Run the app:
       streamlit run app.py
4. The app will open in your browser automatically 
   if not, use the link shown in the terminal : (http://localhost:8501)
